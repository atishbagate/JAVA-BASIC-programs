-------------------- API's and Libraries Notes ----------------------------

API [Application Programming Interface]
It is a collection of Library files which are used to communicate with the peices of softwares programmatically within the same or different application.

Libraries
It is a collection of jar files which comineley form an API.

JAR 
It is a collection of multiples packages.

Package
It is a collection of multiple .java and .class files of different classes and interfaces.
